# EduFormacije

EduFormacije je aplikacija koja omogućuje pretragu osnovnih i srednjih škola, filtriranje i preporuku srednjih škola u Hrvatskoj na temelju zadanih interesa. Aplikacija koristi generativni AI model koji je treniram prema podacima škola za preporuke školskih programa temeljenih na korisničkim interesima.
